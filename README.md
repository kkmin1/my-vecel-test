# my-vecel-test

정적 웹 페이지를 간단히 배포해 보기 위한 소규모 Vercel 테스트 저장소입니다.

## 개요

현재 저장소는 빌드된 `index.html`과 정적 `assets` 폴더 중심으로 구성되어 있습니다. 복잡한 앱보다는 배포 결과 확인과 정적 호스팅 동작 점검에 가까운 실험용 저장소입니다.

## 실행 방법

정적 파일 서버로 확인할 수 있습니다.

```bash
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속합니다.

## 용도

- Vercel 배포 테스트
- 정적 사이트 결과물 확인
- 아주 작은 프론트엔드 실험 배포
